============
Installation
============


Stable release
--------------

To install flake8-precommit-hook, run this command in your terminal:

.. code-block:: console

    $ pip install flake8_precommit_hook

This is the preferred method to install flake8-precommit-hook, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for flake8-precommit-hook can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/kalibrr/flake8_precommit_hook

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/kalibrr/flake8_precommit_hook/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/kalibrr/flake8_precommit_hook
.. _tarball: https://github.com/kalibrr/flake8_precommit_hook/tarball/master
